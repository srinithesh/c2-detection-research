# 🎓 ScholarPath AI

> *"Built because I AM the target user."*

An AI-powered scholarship readiness analyzer that helps 
Indian students identify the gap between their current 
academic profile and profiles of past selected scholars 
for competitive international scholarships.

## 🎯 Problem

Indian students targeting DAAD, Erasmus Mundus, and 
Chevening scholarships have NO way to know:
- How competitive their profile actually is
- What specific gaps exist vs successful applicants  
- What to prioritize in the time they have

Most students find out they weren't competitive 
AFTER rejection. ScholarPath AI tells you BEFORE.

## 💡 Solution

Input your profile → AI compares vs past winners → 
Get personalized gap analysis + roadmap

## 🔍 What It Analyzes

| Profile Area | What We Check |
|-------------|---------------|
| **Academic** | CGPA vs scholarship average, course relevance |
| **Research** | Papers published, conference presentations |
| **Certifications** | Relevant industry certs |
| **Internships** | Duration, relevance, company tier |
| **Languages** | German/Finnish/French for EU scholarships |
| **Extracurriculars** | Leadership, volunteering, social impact |
| **SOP Quality** | AI-powered statement of purpose feedback |

## 🎯 Scholarships Covered

- 🇩🇪 DAAD (Germany)
- 🇪🇺 Erasmus Mundus (EU — multiple programs including CYBERUS)
- 🇬🇧 Chevening (UK)
- 🇫🇮 Finland Government Scholarship
- 🇳🇴 Norway Quota Scheme

## 🛠️ Tech Stack

- Python (Backend logic)
- LLM API (Profile analysis + roadmap generation)
- Streamlit (User interface)
- Pandas (Profile data handling)

## 🚀 How to Run

```bash
git clone https://github.com/srinithesh/scholarpath-ai
cd scholarpath-ai
pip install -r requirements.txt
streamlit run app.py
```
